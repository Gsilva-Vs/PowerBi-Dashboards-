###  Sobre o Repositório

Este repositório reúne meus projetos de dashboards desenvolvidos com **Power BI**, ferramenta líder em Business Intelligence. Aqui, eu transformo dados brutos em insights estratégicos por meio de visualizações interativas, métricas consolidadas e narrativas visuais.

O objetivo é demonstrar minha capacidade de conectar dados, criar modelagens eficientes e entregar análises que realmente apoiam a tomada de decisão.

---

###  Dashboards Disponíveis

| Dashboard | Descrição | Status |
|---|---|---|
|  **Desempenho de Vendas** | Análise completa do desempenho comercial, com indicadores de faturamento, produtos mais vendidos e sazonalidade. |  Finalizado |
|  **Finanças Pessoais** | Dashboard para controle financeiro, categorizando receitas, despesas e apresentando a evolução do saldo e das metas. |  Finalizado |
|  **Análise de Saúde** *(em breve)* | Painel para acompanhamento de indicadores de saúde e bem-estar. |  Em desenvolvimento |
|  **E-commerce Analytics** *(em breve)* | Análise de funil de vendas, ticket médio e comportamento do cliente em uma loja online. |  Planejado |

*   **Limpeza e Transformação dos Dados (ETL)** com o **Power Query**.
*   **Criação de Métricas e Indicadores** (KPIs) usando a linguagem **DAX**.
*   **Modelagem de Dados** para garantir a correta relação entre as tabelas.
*   **Design e UX** focado na experiência do usuário e na clareza da informação.

###  Como Visualizar os Projetos

Para explorar os dashboards:
1.  Faça o download do repositório: `git clone https://github.com/Gsilva-Vs/PowerBi-Dashboards-.git`
2.  Abra os arquivos com a extensão `.pbix` usando o **Power BI Desktop**, que é gratuito.
3.  Interaja com os filtros e visuais para descobrir os insights.
